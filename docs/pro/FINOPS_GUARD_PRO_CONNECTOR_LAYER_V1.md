# FinOps Guard Pro Connector Layer V1

## Purpose

This document defines the Pro connector layer.

---

## Connector families

### Vendor metering
- OpenAI
- Anthropic
- other API vendors

### Cloud billing
- AWS
- Azure
- GCP

### SaaS billing
- vendor invoices
- usage exports
- admin usage reports

### Internal finance systems
- procurement systems
- internal budgets
- internal approval workflows

---

## Role of the connector layer

The connector layer moves the system from:
- manual imported evidence

to:
- live governed financial inputs

---

## Why it matters

Without this layer, evaluation remains episodic.

With this layer, governance becomes continuous.
